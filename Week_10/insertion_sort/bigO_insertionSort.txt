O algoritmo “insertion sort” é um algoritmo simples usualmente utilizado para ordenar por exemplo um array, este ordena um elemento de cada vez.  No melhor cenário possível o array inicial já está ordenado, neste caso o “insertion sort” tem uma “time complexity” de O(n), ou seja, durante cada iteração o primeiro elemento é comparado apenas com os restantes elementos da parte ordenada do array.
Em média e no pior cenário possível tem uma “time complexity” quadrática O(n^2) isto indica que o insertion sort irá escrever para o array O(n^2) vezes, tornando-o pouco pratico para ordenar arrays de grandes dimensões.  Ainda assim é considerado o algoritmo mais eficiente para ordenar arrays até um máximo de cerca de 10 elementos.
